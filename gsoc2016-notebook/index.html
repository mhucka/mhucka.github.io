Logics
Common calculi:
Propositional calculus:
A properly formed statement is a proposition. A proposition has a truth value. If you can’t assign a true value, it’s not a proposition.
Sentential connectives combine simple propositions
And: ∧
Or: ∨
Not: ¬
Implies: →
Predicate calculus:
Propositional calculus is not very useful because we need to be able to speak about objets, postulate relationships, and generalize the relationships
Statements about individuals are predicates.
A predicate is applied to a specific number of arguments and has a value of either true or false when individuals are used as the args.
E.g of a predicate: “is-red” applied to one arg.
Predicate calculus also makes use of qualifiers:
For all: ∀
Exists: ∃
Inference rules for quantifiiers: the use of quantifiers often implies the introduction of 4 inference rules, one for introduction and one for elimination of ea of the 2 qualifiers above
First-order logic:
Predicate calculus alone is often clumsy. To make it easier to use, we can introduce 2 more things:
Functions (operators): they return objects related to their arguments.
E.g.: “father-of”
Equals: X = Y if and only if, for all predicates P, P(X) is equivalent to P(Y)
